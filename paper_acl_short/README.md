# ACL Short Paper Draft

This folder contains a paper draft written to be compatible with the ACL Overleaf template.

Files:

- `main.tex`: main paper source
- `custom.bib`: bibliography entries

Recommended usage:

1. Open the ACL conference template in Overleaf.
2. Replace the default `main.tex` with this file.
3. Upload `custom.bib`.
4. Keep the template-provided `acl.sty` and related files.

Notes:

- The main paper is written as a short paper and is intended to stay within the ACL 4-page body limit.
- `Future Work Proposal` is placed after `\appendix`, so it can be treated as non-main-page supplementary material.
